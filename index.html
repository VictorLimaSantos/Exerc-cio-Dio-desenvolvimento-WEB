Historia da WEB

A historia da WEB se inicia antes mesmo dos primeiros computadores pessoais, no inicio do seculo XX com o desenvolvimento de máquinas mecânicas para cálculos. Tendo commo marco importante o ENIAC, criado em 1946, e dando início a era dos computadores digitais eletrônicos, utilizados principalmente para cálculos científicos e militares.

Nos anos 1950 e 1960, surgiram os mainframes, grandes computadores usados principalmente por governos e empresas.

Com a invenção dos transistores (1947) e, mais tarde, dos circuitos integrados (década de 1960), os computadores ficaram menores, mais rápidos e acessíveis. A década de 1970 trouxe os primeiros microcomputadores para uso pessoal, como o Altair 8800 (1975) e o Apple II (1977). Em 1981, a IBM lançou seu IBM PC.

Nos anos 1980, o crescimento da computação pessoal abriu o caminho para a criação de redes. Isso levou ao desenvolvimento da internet, uma rede global de computadores que evoluiu a partir da ARPANET (1969), uma rede criada pelo Departamento de Defesa dos EUA. Finalmente, em 1989, com a criação da World Wide Web por Tim Berners-Lee, a internet foi transformada em uma plataforma acessível ao público, unindo a comunicação e a informação globalmente.

o desenvolvimento das redes de computadores, como a ARPANET (1969), deu origem à internet. E o Cientista Tim Berners-Lee revolucionou a comunicação digital ao propor a World Wide Web, um sistema de compartilhamento de informações baseado em hipertextos. Em 1990, ele criou o primeiro site e desenvolveu os principais elementos da Web: o protocolo HTTP, a linguagem HTML e o conceito de URLs. A primeira página da Web foi lançada em 1991, com a proposta de facilitar o acesso a documentos científicos.

Nos anos seguintes, a Web cresceu rapidamente, com o surgimento de navegadores gráficos como o Mosaic (1993) e o Netscape Navigator (1994). Em 1995, a linguagem de programação JavaScript foi criada, permitindo interatividade nos sites. A popularização da Web transformou a comunicação, o comércio e a sociedade global, levando à era moderna da internet com plataformas dinâmicas, redes sociais e serviços digitais.

Cliente X Servidor

O sistema Cliente-Servidor é um modelo de arquitetura de rede onde duas partes principais interagem: o cliente e o servidor. O cliente é o dispositivo ou software que faz solicitações, enquanto o servidor é o que processa essas requisições e fornece os recursos ou serviços solicitados.

O cliente é o dispositivo, que faz solicitações, geralmente por meio de um software, como um navegador de internet ou um aplicativo. O servidor, por sua vez, é o sistema que recebe essas solicitações, processa os dados e responde com as informações solicitadas.

Esse modelo é amplamente utilizado na Web. Quando você acessa um site, por exemplo, seu navegador (cliente) envia uma solicitação ao servidor, que hospeda o site. O servidor processa a solicitação e envia os dados de volta ao cliente, que exibe a página para o usuário.

Linguagem de Programação

Uma linguagem de programação é um conjunto de instruções que permite a comunicação entre o programador e o computador. Por meio dessas linguagens, os desenvolvedores escrevem códigos para criar programas, aplicativos e sistemas, controlando o comportamento e a lógica de funcionamento de um software. Linguagens de programação, como JavaScript, Python e Java, têm sintaxes e regras que permitem a criação de algoritmos e a execução de tarefas específicas, desde cálculos simples até funções mais complexas como inteligência artificial.

O HTML (HyperText Markup Language), não é considerado uma linguagem de programação porque ele não executa instruções nem contém lógica. Em vez disso, o HTML é uma linguagem de marcação, usada para estruturar o conteúdo de páginas web. Ele define elementos como títulos, parágrafos, imagens e links, mas não contém funcionalidades para processar dados ou executar ações.

Estrutura basica do HTML

A estrutura básica de um documento HTML segue uma hierarquia de elementos denominados de "TAGS" marcadas por "<>" e o conteúdo dentro desta marcação, as tags organizam e definem o conteúdo de uma página web. a estrutura base do HTML é:

<!DOCTYPE html>
<html>
<head>
<title>
<meta>
<body>

Está seria a estrutura padrão de todo documento HTML, sendo que toda pagina mantera esta mesma estrutura. Além disso grande partes das tags, possum um fechamento, para demarcar onde ela se finaliza, sendo estas utilizadas com </>.

O que são tags

As tags HTML são os elementos fundamentais usados para criar e estruturar documentos em HTML. Elas são responsáveis por definir diferentes tipos de conteúdo em uma página da web, como textos, imagens, links, formulários, e muito mais.

Cada tag é delimitada por colchetes angulares (</>). Em sua forma mais simples, as tags consistem em um elemento de abertura e um elemento de fechamento, com o conteúdo entre elas. A tag de fechamento inclui uma barra ("/") antes do nome da tag.

Principais tags

  • TAG H1 - H6

    Representada pela sintaxe <h1>, e fechada com </h1>, representa o titulo do texto, e demanda o nivel de importancia do titulo indo da hierarquia de 1 até 6, onde 1 é o de maior importanciae o 6 de menor importancia.

  • TAG P

    Faz a marcação de paragrafos na pagina, utiliza a tag <p> e </p>. assim a pagina fica estruturada e melhor identificado onde inicia e termina cada paragrafo.

  • TAG IMG

    Utiliza para anexar imagens a pagina web, refenciando um endereço de links externo ou caminho do reporsitorio na maquina. Utiliza o formato <img src="" alt="">. Diferente de outras tags, o "img" nao possui fechamento, alem disso conta com os Atributos "src" que indica o caminho do arquivo, e "alt" que seria um texto alternativo caso a imagem apresenta algum problema no carregamento.

  • TAG A

    Realiza a aconragem com outros links, tanto internos quanto externos, <a href=""></a>, o href diz o caminho a ser apontado, e o conteudo entre a abertura e o fechamento da tag se tornara o hipertextos de ancora.

    Exemplo: <a href="www.google.com.br"> Este link leva até o google </a>

  • Tag Strong

    Utilizada para estilizar uma parte do texto, deixando-o em negrito, veio para substituir a tag descontinuada <b>, sua sintaxi é <strong>o conteudo entre ela ficara em negrito</strong>,

  • Tag I

    Fara o texto ficar com inclinação italica, substituindo o <em>, a sintaxi utilizada é <i>o conteudo entre ela ficara em italico</i>,

  • Tag U

    Defini uma linha abaixo do conteudo delimitado estilo de marcação <u>underline</u>,o conteudo entre ela tera um underline

  • Tag HR

    Faz uma linha horizontal de uma ponta a outa na pagina, normalimente utilizadas para separar temas em uma pagina <hr>

  • Tag Blockquote

    Utilizada para mostar que este conteudo é uma sitação de outra pessoa, deixa o texto levemente centralizado.

    <blockquote>Isto é uma sitação</blockquote>

  • Tag Mark

    Faz uma marcação em um trecho do texto deixando-o destacado com uma faixa colorida, <mark>estilo um marca texto.</mark>

  • Tag small

    Faz um texto com fonte menor ao resto do conteudo da pagina. <small>Texto pequeno</small>

  • Tag SUP

    Define um trecho do texto para ficar acima dos demais, porém na mesma linha, normalmente utilizado em notas. <sup>Isto é uma nota</sup>

  • Tag SUB

    Similar ao sup, o sub define um texto para ficar abaixo do restante da linha. Define um trecho do texto para ficar acima dos demais, porém na mesma linha, normalmente utilizado em notas. <sub>Isto é outra nota</sub>

Tags de Listas

Em HTML também é muito utilizado o sistemas de Listas, temos dois modelos principais, "listas Ordenadas" e "listas não oredenadas", utilizando respecticamente as tags <ol></ol> e <ul></ul>. Além disso para declarar itens nas listas é utilizado o comando <li></li> "List Iten"

Em caso de Listas ordenadas, elas segiram padrões que indiquem uma sequencia (ex.: "123", "abc" ou "I,II,III").

  1. item 1
  2. item 2
  3. item 3

Para as listas não ordenadas, como a ordem não é importande, pode ser feitas com simbolos como circulos, qualdrados, traço e etc.

  • item 1
  • item 2
  • item 3

É possivel estilizar os marcadores atrves de atributos direto na tag ou via CSS.

Outras tags

Há inumeras outras tags disponiveis no HTML além das ja abordadas acima, como por exemplo <font>, <del>, <abbr>, <table> etc.

Abixo uma pequena explicação sobre cada:

Font é utilizada para definir atributos a um texte fonte, o tamanho e a cor. No entanto, essa tag está obsoleta e não é recomendada para uso em HTML5, pois a formatação de texto deve ser feita preferencialmente através de CSS.

O Del, indica que um trecho de texto foi removido ou deletado do conteúdo. É geralmente exibido com uma linha atravessada, sendo útil para mostrar alterações em documentos.

abbr representa uma abreviação ou acrônimo, podendo incluir um atributo title para fornecer a forma completa ou uma descrição do termo. Isso melhora a acessibilidade e compreensão do conteúdo. Para visualizar a descrição basta repousar o mouse sobre o conteudo

Quanto a tag Table, ela é utilizada para a criação de tabelas, onde em conjunto com a tag td e tr compoem as linhas e colunas da tabela.

Atributos de tags

Os atributos de tags HTML são propriedades adicionais que fornecem informações extras sobre um elemento HTML. Eles são usados para modificar o comportamento ou a aparência das tags, permitindo personalizar ainda mais os elementos de uma página.

Os atributos são inseridos dentro da tag de abertura e consistem de um nome e um valor separados por um sinal de igual (=). O valor é colocado entre aspas.

Exemplo: <a href="www.google.com.br"> Este link leva até o google </a>

Neste exemplo, o atributo href define o link para o qual o usuário será direcionado ao clicar no texto.

Principais atributos comuns:

  • href (em <a>): Define o destino de um link.
  • src (em <img>): Especifica o caminho da imagem a ser exibida.
  • alt (em <img>): Fornece um texto alternativo caso a imagem não carregue.
  • id Atribui um identificador único a um elemento.
  • class Define uma ou mais classes para um elemento, usadas para estilização com CSS.
  • style Aplica estilos diretos ao elemento, como cor ou tamanho de fonte.

Os atributos permitem personalizar e controlar como os elementos HTML interagem e se comportam, melhorando tanto a funcionalidade quanto a acessibilidade da página.
